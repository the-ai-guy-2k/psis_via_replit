# Using PSIS to Develop Pitchers

How coaches can apply EABR thinking to long-term pitcher development — with or without software.

---

## Development Is a Trend, Not an Outing

One bullpen session does not define a pitcher. One bad start does not end a career.

PSIS session summaries exist so coaches can ask:

- Is EABR delta improving over weeks?
- Is the EABR fraction stable or volatile?
- Are walk bad units decreasing?
- Are strikeout good units increasing?

Traditional stats will catch up later. **Process trends often appear in EABR first.**

---

## Bullpen Planning

### Before the pen

| Question | PSIS-informed approach |
|----------|------------------------|
| What are we working on? | Sequences that produced bad units last outing |
| What success looks like | Positive delta in live ABs or simulated game situations |
| What to log | Same EABR classifications as games — build habit |

### After the pen

Review at-bats, not just "how it felt." Even informal logging trains the classification muscle staff will use in games.

---

## Game Review

### Same night (10 minutes)

1. **Session delta** — Did we win more at-bats than we lost?
2. **Inning deltas** — Which innings tell the story?
3. **Damage** — Where did RBI bad units cluster?
4. **LOB** — Did we strand runners or let innings snowball?
5. **Sequences** — Any pattern in best/worst at-bats?

### Next day (with video)

Match EABR log to video. Ask:

- Did good units come from execution or luck?
- Did bad units come from mistakes or good hitting?
- What sequence should we repeat or retire?

---

## Confidence

Pitchers lose confidence when only runs matter.

EABR gives coaches language to rebuild it:

| Situation | Coach script |
|-----------|--------------|
| Runs scored, positive delta | "You won more fights than you lost — the damage was clustered, not spread." |
| Negative delta, low ERA | "The line looks okay but we were behind too much — let's fix counts." |
| High LOB inning | "That's pitching — you competed with traffic." |

Confidence built on **observable wins** (good units) is more durable than confidence built on outcomes alone.

---

## Trend Analysis

| Trend | Possible coaching action |
|-------|--------------------------|
| Rising session delta | Maintain approach; add difficulty |
| Falling fraction | Review sequences; check walk rate |
| High RBI bad units | Location in damage counts; pitch selection with RISP |
| High LOB good units | Reinforce competing in jams — skill is translating |
| Volatile inning deltas | Focus on consistency; mental game between innings |

---

## Individual Coaching Conversations

EABR makes 1-on-1 meetings specific:

> "Over your last four sessions, your delta is +6, +2, −1, +4. The −1 session was three walks in one inning — not contact. Let's work first-pitch strikes and the sequence you use behind in the count."

Compare to:

> "You're throwing fine, keep going."

The first conversation develops. The second comforts without direction.

---

## Staff Alignment

When multiple coaches work with the same pitcher:

- Shared EABR definitions reduce mixed messages
- Session summaries give catching coaches, pitching coaches, and coordinators the same artifact
- Disagreements surface early (classification, not personality)

---

## Without the Software

Coaches can apply this framework with a notebook:

| Column | Content |
|--------|---------|
| Inning | 1–9 |
| Sequence | FB-SL-CH |
| Outcome | K, BB, 6-3, etc. |
| Good | 1 or 0 |
| Bad | 1 + RBI |
| LOB | On 3rd out only |

PSIS automates math and aggregation. The **philosophy works on paper.**

---

## What Not to Do

| Mistake | Why |
|---------|-----|
| Optimize EABR instead of pitching | Metric becomes the goal |
| Punish negative delta without context | Ignores luck and defense |
| Skip logging bad innings | Survivorship bias in trends |
| Compare pitchers by delta alone | Role, competition, and sample size differ |
